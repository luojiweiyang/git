### Claude

克隆 GitHub 私仓到本地，步骤如下：

## 1. 确认你有访问权限

确保你的 GitHub 账号对该私仓有读取权限。

## 2. 选择认证方式

**方式一：HTTPS + Personal Access Token（推荐）**

1. 在 GitHub 生成 Token：`Settings` → `Developer settings` → `Personal access tokens` → `Generate new token`，勾选 `repo` 权限
2. 克隆时用 Token 代替密码：
   ```bash
   git clone https://github.com/你的用户名/仓库名.git
   ```
   提示输入用户名和密码时，密码处填 Token

**方式二：SSH Key**

1. 检查本地是否已有 SSH Key：
   ```bash
   ls ~/.ssh/id_*.pub
   ```
2. 如果没有，生成一个：
   ```bash
   ssh-keygen -t ed25519 -C "你的邮箱"
   ```
3. 复制公钥内容：
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```
4. 将公钥添加到 GitHub：`Settings` → `SSH and GPG keys` → `New SSH key`
5. 克隆：
   ```bash
   git clone git@github.com:你的用户名/仓库名.git
   ```

**方式三：使用 GitHub CLI（最方便）**

```bash
# 安装
brew install gh

# 登录（会打开浏览器授权）
gh auth login

# 克隆
gh repo clone 你的用户名/仓库名
```
